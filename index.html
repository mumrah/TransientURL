TransientURL by mumrah

A one-time use URL shortener.

TransientURL (turl) is a temporary URL shortener. The generated transient URLs are good for one use only. The original motivation behind this was to have a way for my EC2 startup scripts to download secure content from S3 without having the AWS credentials visible. If you look at the source code, I've created an abstract "Key" class. This is to allow developers to implement their own key storage mechanism. I initial built it to use serialized python files stored on the local disk (like PHP sessions). Plans exist to allow SimpleDB as a distributed storage mechanism. There are many possibilities for extending this base Key class (relational db, flat file db, memcache, etc). The use of cherrypy was for quick development purposes only. Any Python framework would do just fine (or in fact, simple mod_python would work).

Dependencies

python2.5, python-cherrypy3, python-boto

Install

On a debian based system, install cherrypy and boto like "sudo apt-get install python-cherrypy3 python-boto". On other systems, these libraries will need to be built from source. Future extensions to the Key storage mechanism may require additional installs.
